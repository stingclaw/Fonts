How to Use

This repository contains fonts for your computer, not your website (i.e. you can use them in Microsoft Word, Pages, etc.). Instructions:

Version 1: Download the whole enchilada, and install only some fonts

  1. Download a .zip or tarball of the repository (on site, or GitHub)
  2. Selectively install the fonts you like
  3. Delete the .zip or tarball from your Downloads folder when you're done (to save space)

Version 2: Selectively download from GitHub

  1. If you view a raw font file, usually it will try to download.
  2. Selectively view the raw files of each font file you want to download
  3. Install the fonts by clicking on them/opening them.
