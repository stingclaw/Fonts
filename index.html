A collection of downloadable font files, categorized by type.*

How to Use

This repository contains fonts for your computer, not your website (i.e. you can use them in Microsoft Word, Pages, etc.). Instructions:

Version 1: Download the whole enchilada, and install only some fonts

  • Download a .zip of the repository (on site, or GitHub)
  • Selectively install the fonts you like
  • Delete the .zip from your Downloads folder when you're done (to save space)

Version 2: Selectively download from GitHub

  • If you view a raw font file, usually it will try to download.
  • Selectively view the raw files of each font file you want to download
  • Install the fonts by clicking on them/opening them.
